## Text Objects

tldr;

`[command][(i)nner|(a)][text object]`
There are various text object, and you can put more through plugins.

#### Normal texts


- `w` for word
- `s` for sentence
- `p` for paragraph

#### Code
- `({[` for brackets and stuff
- `t` for html tag content
- `<>` for html tags
- `"'[backticks]`
-
